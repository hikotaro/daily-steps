# 歩数登録アプリ (Step Tracker)

毎日の歩数を簡単に記録・管理できるシンプルなReactアプリケーションです。

## 🚀 機能 (Features)

- **歩数の登録**: 入力フォームから歩数を入力し、ボタン一つで記録を追加できます。
- **自動日付付与**: 登録時の日付（YYYYMMDD形式）が自動的に計算され、データに紐づきます。
- **歩数履歴の表示**: これまでに記録した毎日の歩数データを一覧で確認できます。

## 🛠️ 使用技術 (Tech Stack)

- React (Hooks: `useState`)
- JavaScript / JSX
- CSS (`App.css` によるスタイリング)

## 📂 ディレクトリ構成 (Folder Structure)

主なファイル構成は以下の通りです。

```text
src/
├── App.jsx        # メインコンポーネント（状態管理・フォーム・リスト表示）
├── App.css        # アプリケーション全体のスタイル
└── components/
    └── Step.jsx   # 各歩数データ（1日分）を表示するためのコンポーネント
```

## 💻 起動方法 (Getting Started)

ローカル環境でこのプロジェクトを動かすための手順です。

1.プロジェクトのルートディレクトリに移動します。

2.依存パッケージをインストールします。

```bash
npm install
```

3.開発サーバーを立ち上げます。

```bash
npm run dev
```

4.ブラウザでターミナルに表示されたローカルURLにアクセスして動作を確認します。
